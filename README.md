# Racket-N-Queens-Solver
This is a fun little project from the 1st semester of my bachelor's, written in Racket (a Scheme dialect).

## What is the N-Queens Problem?
The N-Queens problem is a classic combinatorial search problem:
Place N queens on an N × N chessboard such that no two queens attack each other.

This means:
No two queens share the same row
No two queens share the same column
No two queens share the same diagonal

## To run this Solver
[Download Racket](https://racket-lang.org) and run the file.

Have fun ;)
